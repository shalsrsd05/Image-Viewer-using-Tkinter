# Image Viewer using Tkinter

A simple and user-friendly **Image Viewer desktop application** built using **Python** and **Tkinter**. This project allows users to view images from a local folder and navigate through them using **Next** and **Previous** buttons.


##  Features

* View images stored on your local system
* Supports common formats: **JPG, PNG, BMP**
* Easy navigation using **Next** and **Previous** buttons
* Clean and simple graphical user interface
* Beginner-friendly Python GUI project

##  How to Run the Project

1. Clone this repository:

```bash
git clone https://github.com/your-username/image-viewer-tkinter.git
```

2. Navigate to the project folder:

```bash
cd image-viewer-tkinter
```

3. Install required library:

```bash
pip install pillow
```

4. Run the application:

```bash
python image_viewer.py
```
